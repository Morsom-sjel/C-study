   2、次方函数：pow（a，b）；
   *3、保留小数点后2位：输出里是“%.2lf”
   4、double型输入是“%lf”*
   5、最后输出是“max”**

**我的代码：**

```
#include<stdio.h>
#include<math.h>
int main()
{
	int N;
	double a,b,max=0,j;
	int i;

	scanf("%d",&N);
	
	for(i=0;i<N;i++)
	{
		scanf("%lf %lf",&a,&b);
		j=sqrt(a*a+b*b);
		if(j>max)
		{
			max=j;
		}
	}
	printf("%.2lf",max);
	return 0;
}
```
