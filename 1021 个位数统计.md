**1、字符‘0’对应的ASCII码是48
        所以c-‘0’就是转换为十进制位数
    2、!='\0'意思就是运行到字符串结尾时结束**

**我的代码：**

```
#include<stdio.h>

int main()
{
	char N[1001];
	int i,j,k;
	int a[100]={0};
	scanf("%s",&N);
	for(i=0;N[i]!='\0';i++)
	{
		j=N[i]-'0';
		a[j]++;
	}
	
	for(j=0;j<=9;j++)
	{
		if(a[j])
		{
			printf("%d:%d\n",j,a[j]);
		}
	}
	return 0;
}
```
